個人研究：読書支援システムの開発

読書離れを解消するための支援システムとして考案しました。 小説の文章を段落ごとに分割し、大規模言語モデルで拡散モデルに入力するためのプロンプトに変更させ、そのプロンプトを既存の拡散モデルに入力することでVR空間画像を確立しました。 今後は画像切り替えタイミングの検討や画像間の登場人物の統一化を検討しています。
